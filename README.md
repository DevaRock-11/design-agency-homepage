# Design Agency Homepage

This project is a responsive Design Agency landing page built using Next.js.

## Tech Stack

- Next.js 14
- React
- Tailwind CSS
- Framer Motion (for animations)

## Features

- Hero section with CTA
- Services section with cards
- Portfolio grid with hover effects
- Contact form
- Fully responsive design
- Image optimization using Next/Image

## Setup Instructions

1. Clone the repository

git clone https://github.com/yourusername/design-agency-homepage.git

2. Install dependencies

npm install

3. Run development server

npm run dev

4. Open in browser

http://localhost:3000
